ÉTAPE 0 — DÉMARRAGE SUR LE NOUVEAU PC (à faire en premier)
  [ ] Installer Git : https://git-scm.com/download/win
  [ ] Cloner le repo : git clone https://github.com/edson-gpu/vigiparl.git
  [ ] Installer Docker Desktop : https://www.docker.com/products/docker-desktop/
        (ou winget install --id Docker.DockerDesktop -e en admin)
        IMPORTANT : redémarrer le PC après l'installation de Docker
  [ ] Installer Node.js 20+ : https://nodejs.org/
  [ ] Copier le fichier .env dans backend/ avec les credentials PISTE ci-dessus
  [ ] Vérifier que Docker fonctionne : docker --version

ÉTAPE 1 — LANCER L'ENVIRONNEMENT LOCAL
  [ ] cd vigiparl
  [ ] docker compose up   ← démarre PostgreSQL + Redis + backend + workers
  [ ] cd frontend && npm install   ← installe les dépendances Node.js
  [ ] cd backend && alembic upgrade head   ← crée les tables en BDD
  [ ] Vérifier l'API : http://localhost:8000/docs (Swagger auto-généré)
  [ ] Vérifier le frontend : http://localhost:3000
  [ ] Ajouter ANTHROPIC_API_KEY dans .env (clé Claude API à créer/récupérer)
        → https://console.anthropic.com/
